Vigilar

A security companion that runs entirely on your device. No cloud, no account, no data collection.

What is Vigilar?

Vigilar is a security app for macOS and Android. It monitors your system for threats, alerts you when your data appears in breaches, and includes an AI companion you can chat with about your security — all running locally on your device.

🔒 On-device first
Vigilar never sends your data to any server. The AI model runs on your device. Breach checks use k-anonymity hashing so your email is never transmitted in full.

Key Features

  • Security Dashboard — Gatekeeper, SIP, FileVault, Firewall, XProtect status at a glance
  • Breach Monitor — Checks your email against known breach databases (HIBP API)
  • AI Companion Chat — On-device 12B model, no cloud, no API calls
  • Threat Detection — Suspicious processes, launch agents, open ports, installed apps audit
  • Security Score — Weighted assessment with actionable recommendations
  • Identity Theft Protection — Offline heuristic analysis of exposure vectors
  • ARCZT Vault Link — Deep-link bridge to ARCZT Personal Vault for breach-to-safe matching

Platforms

  • macOS — DMG installer, Apple Silicon (arm64), Electron + React Native
  • Android — APK installer, React Native, offline-first
  • iOS — Planned (same codebase, App Store submission pending)

How It Works

Vigilar runs a custom-trained 12B parameter language model directly on your device. The model (Ember Prime) is quantized to 4-bit (7.4GB) and loads into RAM at startup. All chat, threat analysis, and security reasoning happens locally — no network connection required after installation.

Breach monitoring uses the Have I Been Pwned API with k-anonymity: only the first 5 characters of your email's SHA-1 hash are sent, and Vigilar filters the results locally. Your full email address never leaves your device.

Technical Specs

  • Model: Ember Prime v8c, 12B parameters, Q4_K_M quantization (7.4GB)
  • Inference: llama.cpp (desktop), MLC (mobile)
  • Framework: React Native + TypeScript + Electron
  • Storage: License heartbeat only (SecureStore/Keychain). No chat persistence, no cloud sync.
  • Bundle size: macOS DMG ~640MB, Android APK ~120MB
  • Min OS: macOS 12+, Android 8+ (API 26)
