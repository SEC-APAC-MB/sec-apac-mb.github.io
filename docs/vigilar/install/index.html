🔥 Vigilar

Installation

Install Vigilar on macOS or Android. No account required, no cloud setup.

macOS (Apple Silicon)

Requirements

  • macOS 12.0 or later
  • Apple Silicon (M1, M2, M3, M4)
  • 16GB RAM minimum (8GB used by AI model)
  • 1GB free disk space

Steps

  1. Download the Vigilar-x.x.x-arm64.dmg file
  2. Open the DMG and drag Vigilar to Applications
  3. Launch Vigilar from Applications
  4. On first launch, macOS may warn about an unidentified developer. Right-click → Open to bypass.
  5. Vigilar loads the AI model (7.4GB) into RAM — this takes ~10 seconds
💾 Model bundled
The AI model is included in the DMG. No additional download required after installation.

Android

Requirements

  • Android 8.0 (API 26) or later
  • 8GB RAM minimum
  • 500MB free storage

Steps

  1. Download the Vigilar-x.x.x.apk file
  2. Enable "Install from unknown sources" in Android Settings → Security
  3. Open the APK file to install
  4. Launch Vigilar
📱 Mobile model
Android uses a smaller model quantization (4-bit, ~4GB) to fit within mobile memory constraints.

iOS

iOS support is planned. The codebase is shared with Android via React Native. App Store submission is pending.

Post-Install Setup

  1. License activation — Enter your license key (emailed after purchase). Free tier requires no key.
  2. Breach monitor — Enter your email to check against known breaches. Uses k-anonymity — your full email never leaves the device.
  3. ARCZT Vault link (optional) — If ARCZT Vault is installed, tap "Open in ARCZT Vault" on any breach alert to link entries.

Uninstall

  • macOS: Drag Vigilar from Applications to Trash. No leftover files — Vigilar stores nothing outside the app bundle except the license in Keychain.
  • Android: Settings → Apps → Vigilar → Uninstall.
🔒 Clean removal
Vigilar stores only a license heartbeat in SecureStore/Keychain. No chat history, no cached data, no config files. Uninstalling removes everything.

Troubleshooting

Model won't load

Ensure you have enough free RAM. Close other memory-intensive apps. The model needs ~8GB on desktop, ~4GB on mobile.

macOS "unidentified developer" warning

Right-click the app → Open → Open Anyway. This is expected for DMG installs outside the Mac App Store.

Breach monitor shows no results

Ensure you have an internet connection. Breach checks use the HIBP API which requires network access. All other features work offline.
