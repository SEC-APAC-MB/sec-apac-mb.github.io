Installation

Install ARCZT Vault as a desktop app, mobile app, or self-hosted service.

Self-Hosted (Docker Compose)

Requirements

  • Docker 20.10+ and Docker Compose 2.0+
  • 2GB RAM minimum
  • 1GB free disk space

Steps

  1. Clone the repository: git clone https://github.com/SEC-APAC-MB/ARCZT-Personal-Vault.git
  2. Navigate to the project: cd ARCZT-Personal-Vault
  3. Copy the environment file: cp .env.example .env
  4. Edit .env — set your SECRET_KEY, POSTGRES_PASSWORD, and REDIS_PASSWORD
  5. Start the stack: docker compose up -d
  6. Access the web UI at http://localhost:3000
🆓 Free and unlimited
Self-hosting is free with all features unlocked. Community support only.

Desktop App (Planned)

macOS

  • Download ARCZT-Vault-x.x.x-arm64.dmg
  • Open DMG and drag to Applications
  • Launch and create your master password

Windows

  • Download ARCZT-Vault-Setup-x.x.x.exe
  • Run installer and follow prompts

Mobile Apps

Android

  • Download ARCZT-Vault-x.x.x.apk
  • Enable "Install from unknown sources"
  • Open APK to install

iOS

Planned — Swift implementation in development.

Browser Extension

Chrome / Firefox

  1. Install from Chrome Web Store or Firefox Add-ons (pending review)
  2. Click the ARCZT icon in your browser toolbar
  3. Enter your vault URL (self-hosted) or connect to desktop app
  4. Autofill works on login forms — the extension never exposes your safe

Post-Install Setup

  1. Create master password — Choose a strong password. This derives your encryption key. There is no recovery.
  2. Set up MFA (Pro tier) — Add TOTP authenticator for vault access
  3. Add entries — Manually add credentials or import from CSV (1Password, Bitwarden, LastPass formats supported)
  4. Enable Vigilar link (optional) — If Vigilar is installed, breach alerts will deep-link to ARCZT

Import from Other Managers

ARCZT supports CSV import from:

  • 1Password
  • Bitwarden
  • LastPass
  • KeePass (CSV export)
  • Chrome / Firefox (built-in password manager export)

Backup

Your encrypted safe file can be exported at any time. The export is encrypted — it cannot be read without your master password. Store it safely.

⚠️ Master password
If you lose your master password, your data is unrecoverable. This is the trade-off of zero-knowledge encryption. Keep a secure backup.
